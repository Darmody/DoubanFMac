## Features

1. play/pause songs on 私人 channel. [`Ctrl+Alt+p`]
2. like songs. [`Crtl+Alt+l`]
3. dislike songs. [`Ctrl+Alt+u`]
4. skip songs. [`Ctrl+Alt+n`]
5. ban songs. [`Ctrl+Alt+b`]
